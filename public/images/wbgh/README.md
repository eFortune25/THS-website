# WBGH Logo Files

Place your WBGH logo files in this directory:

Required files:
- wbgh-logo.png or wbgh-logo.svg (main color logo)
- wbgh-logo-white.png or wbgh-logo-white.svg (white version for dark backgrounds)

Optional:
- wbgh-favicon.ico (for browser favicon)
- wbgh-logo.png at 192x192px for high-res displays

After adding logo files, the WBGHLogo component will be updated to use them.
